# First Chord Brain Vault

This is an Obsidian vault for First Chord Music School operations.

## Structure

```
vault/
├── Onboarding/         # Auto-generated onboarding records
│   └── YYYY-MM-DD-student-name.md
│
├── Students/           # Student profiles (future)
│
├── Meetings/           # Weekly meeting notes (future)
│   ├── Monday/
│   ├── Wednesday/
│   └── Friday/
│
├── Projects/           # Active projects (creative corner, etc)
│
├── Templates/          # Reference templates
│   ├── Welcome-Message.md
│   ├── Soundslice-Setup.md
│   └── Missed-Call-Response.md
│
└── Reference/          # Reference data
    └── Pricing-and-Templates.md
```

## Usage

1. Open this folder as an Obsidian vault
2. Onboarding records are auto-generated by the Brain CLI
3. Use for tracking students, meetings, and projects

## Auto-Generated Content

When you run `python brain.py onboard "Student Name"`, a new markdown file is created in `Onboarding/` with:
- Student details
- Lesson information
- WGCS checklist
- Timestamp

This keeps a permanent record of all onboardings.
